## 🛒 Retail FreshMart

Retail FreshMart is a web-based retail management system designed to streamline store operations such as sales tracking, inventory monitoring, and performance analytics. 

It provides a clean dashboard interface to help store managers make data-driven decisions efficiently.

🔗 Live Demo    : [https://retail-freshmart.netlify.app/]  
🔗 GitHub Repo  : [https://github.com/Madhan-03/retail-freshmart]  

---

## 🚀 Features

- 📊 Dashboard with key performance indicators (Sales, Profit, Stock Alerts)
- 📦 Inventory management with low-stock alerts.
- 📈 Sales analytics by category and time.
- 🔍 Filter and search functionality for better insights.
- 🧾 Organized data visualization for decision making.

---

## 🛠️ Tech Stack

- Frontend       : HTML, CSS, JavaScript
- Visualization  : Charts / Dashboard components
- Hosting        : Netlify

---

## 🎯 Objective

The goal of this project is to simulate a real-world retail management system that helps store managers monitor performance, reduce stock issues, and improve sales strategies.
